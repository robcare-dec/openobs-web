# Security Policy

## Supported versions

This repository is a public website and is maintained on the `main` branch.

## Reporting a vulnerability

Please do not report security issues in public GitHub issues.

If you discover a vulnerability related to this repository, email the maintainer or organization contact privately with:

- a description of the issue
- steps to reproduce it
- the affected area
- any suggested remediation

We will review the report as quickly as possible and work on a fix before public disclosure when appropriate.
